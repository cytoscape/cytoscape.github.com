Report a Bug

Please use this form to report reproducible Cytoscape bugs. It is helpful if you can confirm that the bug is reproducible on another computer.

Please use the Cytoscape helpdesk instead to ask general questions about Cytoscape, including questions about Cytoscape installation problems and installing apps.

Tips for writing a good bug report:

  • Add your platform information
    • Cytoscape version
    • OS (Mac/Windows/Linux)
    • Java (JVM/JDK) version
  • Be specific - How can you reprocuce it?
  • Optional: sample data to reproduce the bug
