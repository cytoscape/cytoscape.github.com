Report a Bug

Use this form to report Cytoscape bugs. Please include enough information to allow someone else to reproduce the bug. It is also helpful if you can confirm that the bug is reproducible on another computer.

For general questions about Cytoscape installation and usage, visit our user documention page.

Bug Report Tips:

  • Add your platform information
    • Cytoscape version
    • OS (Mac/Windows/Linux)
    • Java (JVM/JDK) version
  • Be specific - How can you reprocuce it?
  • Optional: sample data to reproduce the bug
